## 📝 Cara Menggunakan Template

### 1. **Gunakan File TEMPLATE.md**
   - Salin template
   - Isi sesuai materi yang dipelajari
   - Simpan dengan nama deskriptif

### 2. **Komponen Template (Maksimal 5)**
   - **Judul & Tujuan** - Apa yang dipelajari
   - **Konsep Utama** - Penjelasan singkat
   - **Sumber Belajar** - Link YouTube, artikel, dokumentasi
   - **Catatan Penting** - Key points & takeaway
   - **Latihan/Praktik** - Lab atau eksperimen

### 3. **Format Penamaan File**
   ```
   TAHAP-[1/2]_[KATEGORI]_[SUBTOPIK].md
   
   Contoh:
   - TAHAP-1_NETWORKING_IP-ADDRESS.md
   - TAHAP-1_SECURITY_CIA-TRIAD.md
   - TAHAP-2_RED-TEAM_NMAP.md
   ```

## ✅ Checklist Sebelum Upload

### Safe to Publish ✔️
- Catatan teori & konsep
- Cheatsheet & command reference
- Materi pembelajaran
- Diagram & visual
- CTF writeup
- Lab pada VM pribadi
- Script pembelajaran

### ⚠️ JANGAN Publish ❌
- Password / API Key
- Private Key / SSH Key
- VPN Config
- IP publik / jaringan asli
- Hasil scan jaringan rumah/kantor
- Data perusahaan
- File .env
- Data sensitif apapun

## 🚀 Tips Belajar Efektif

1. **Dokumentasi Real-Time**
   - Tulis saat belajar, jangan menunggu selesai
   - Catat pertanyaan yang muncul

2. **Multi-Source Learning**
   - Lihat 2-3 sumber berbeda untuk 1 topik
   - Bandingkan perspektif berbeda
   - Catat mana yang paling mudah dipahami

3. **Practice-Driven**
   - Setiap topik = ada lab/praktik
   - Link ke repo lab pribadi atau screenshot hasil
   - Dokumentasikan hambatan & solusi

4. **Review Berkala**
   - Setiap selesai subtopik, review catatan
   - Highlight mana yang masih kurang paham
   - Revisi catatan setelah review

5. **Build Portfolio**
   - Kumpulkan lab writeup
   - Share CTF solution (tanpa spoiler penuh)
   - Showcase di GitHub untuk calon employer
